# Spam Detector 📨

An intelligent web application that uses Natural Language Processing (NLP) to classify messages into **Spam** or **Not Spam**.

---

## 🚀 Features

-   **Real-time Classification**: Instantly categorizes any message pasted into the web interface.
-   **NLP-Powered Engine**: Utilizes sophisticated NLP techniques for text preprocessing and feature extraction.
-   **Machine Learning Model**: Employs a **Scikit-learn Classifier** trained on the SMS Spam Collection dataset.
-   **Interactive UI**: A sleek and modern user interface built with Flask and Tailwind CSS.

---

## 🛠️ Technology Stack & Concepts

-   **Backend**: **Python**, **Flask**
-   **Machine Learning**: **Scikit-learn**, **Pandas**, **Gensim**, **NLTK**
-   **Frontend**: **HTML**, **Tailwind CSS**, **JavaScript**
-   **Core NLP Concepts**:
    -   **Text Preprocessing**: Implemented techniques like tokenization, stop-word removal, and lemmatization to clean and prepare the textual data.
    -   **Feature Extraction**: Used **Word2Vec** to convert the processed text into meaningful numerical vectors (word embeddings) that the machine learning model can understand.
